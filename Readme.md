# Medicine Reminder System

A simple Python project that reminds users to take their medicines at scheduled times.

## Project Description
This program allows users to enter medicine names and reminder times.  
The system stores the medicines in a file and continuously checks the current time to notify the user when it is time to take the medicine.

## Features
- Add medicine name and reminder time
- Store medicines in a file
- Display reminder when the time matches
- Supports multiple medicines

## Technologies Used
- Python
- File Handling
- Datetime Module
